# Pharmacy_Sales_Analysis

## Overview
Analysis of pharmacy sales data using SQL Server and Power BI.

## Dataset
- 2106 records
- 13 columns
- Sales data from 2014 to 2019

## Tools Used
- SQL Server Management Studio (SSMS)
- Power BI

## Analysis
- Total Sales by Drug
- Anti-inflammatory Comparison: M01AB vs M01AE
- Painkiller Comparison: N02BA vs N02BE
- Total Sales by Year
- Total Sales by Month
- Total Sales by Weekday

## Dashboard
Power BI dashboard with 2 pages:
- Overview & Drug Analysis
- Sales Trends

## Key Findings
- N02BE (Paracetamol) is the top selling drug with 63,005 total sales
- M01AB (Aceclofenac) outsells M01AE (Ibuprofen)
- N02BE (Paracetamol) outsells N02BA (Aspirin) by 8x
- Saturday is the busiest day for sales
- January has the highest sales — winter season
- 2016 was the best year for sales
